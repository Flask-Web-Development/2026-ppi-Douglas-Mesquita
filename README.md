Estudante: Douglas Mesquita (INFO3V) 

Matéria: Programação para Internet

## CRUD Carrinho
Este sistema consiste em no gerenciamento de itens em um Carrinho de compras junto com o Cadastro de novos usuários com verificação de disponibilidade de nome, login e encerramento de sessão

#### Gestão de Compras:
Listagem completa de produtos com preços e descrições detalhadas

Adição de itens ao carrinho com definição de quantidades

Visualização de itens selecionados com cálculo automático de subtotal e valor total da compra

Opções para alterar quantidades, remover itens específicos ou limpar todo o carrinho

Persistência: O carrinho permanece vinculado à conta do usuário durante toda a sua existência no sistema

## Flask Tutorial 
O desenvolvimento seguiu rigorosamente a estrutura modular e as práticas recomendadas pelo tutorial oficial do Flask

Como a utilização de Application Factory e Blueprints para organizar as rotas de autenticação (auth.py) e da loja (shop.py)
